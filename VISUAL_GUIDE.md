# ➕ КАК ДОБАВИТЬ АНКЕТУ В PROFILES.JSON

```
ШАГ 1: Файл profiles.json выглядит так
┌─────────────────────────────────────┐
│ {                                   │
│   "profiles": [                     │
│     {                               │
│       "id": 0,                      │
│       "name": "Анна",               │
│       ...                           │
│     },  ← запятая есть              │
│     {                               │
│       "id": 1,                      │
│       "name": "Мария",              │
│       ...                           │
│     }   ← запятой НЕТ (последняя)   │
│   ]                                 │
│ }                                   │
└─────────────────────────────────────┘

ШАГ 2: Поставь курсор СЮДА
┌─────────────────────────────────────┐
│ {                                   │
│   "profiles": [                     │
│     {                               │
│       "id": 0,                      │
│       ...                           │
│     },                              │
│     {                               │
│       "id": 1,                      │
│       ...                           │
│     },  ← ДОБАВЬ запятую            │
│ █   ← И СЮДА вставь новую анкету    │
│   ]                                 │
│ }                                   │
└─────────────────────────────────────┘

ШАГ 3: Вставь шаблон
┌─────────────────────────────────────┐
│     },                              │
│     {                               │
│       "id": 2,                      │
│       "name": "Виктория",           │
│       "city": "Москва",             │
│       "age": 23,                    │
│       "height": 168,                │
│       "weight": 54,                 │
│       "appearance": "Блондинка",    │
│       "phone": "+79099999999",      │
│       "telegram": "vika_model",     │
│       "photos": [                   │
│         "ВСТАВЬ_ССЫЛКУ_1",          │
│         "ВСТАВЬ_ССЫЛКУ_2"           │
│       ]                             │
│     }  ← запятой НЕТ (теперь это    │
│          последняя анкета)          │
│   ]                                 │
└─────────────────────────────────────┘

ШАГ 4: Замени ВСТАВЬ_ССЫЛКУ_1 на настоящую ссылку из ImgBB
```

# 🎯 КОНТРОЛЬНЫЕ ТОЧКИ

## ТОЧКА 1: После установки Python
```
Команда в терминале:
python --version

Должно показать:
Python 3.12.0 (или другая версия)

❌ Если ошибка - переустанови Python с галочкой "Add to PATH"
```

## ТОЧКА 2: После установки aiogram
```
Команда:
pip show aiogram

Должно показать:
Name: aiogram
Version: 3.15.0
...

❌ Если "not found" - установи: pip install aiogram
```

## ТОЧКА 3: После вставки токена
```
Открой config.py

Должно быть:
BOT_TOKEN = "1234567890:ABCdefGHI..."
            ↑                    ↑
          кавычка             кавычка

❌ Если без кавычек - будет ошибка!
```

## ТОЧКА 4: После запуска бота
```
В терминале должно быть:
INFO - Бот запущен!
INFO - Start polling

❌ Если ошибка - читай текст ошибки
```

## ТОЧКА 5: После команды /start в Telegram
```
Бот должен ответить кнопками

❌ Если не отвечает:
- Проверь работает ли бот (терминал открыт?)
- Правильный ли username бота?
- Токен верный?
```
